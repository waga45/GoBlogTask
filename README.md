# GoBlog 博客管理系统

### GoBlogServer - 后端服务
**GoBlogServer** 是博客管理系统后台，基于gin的轻mvc架构。提供完整的博客管理API接口，包括用户管理、文章管理、评论管理等功能。

### web - 前端管理系统
**web** 是博客管理的前端web，基于vue2的管理系统。提供直观的用户界面，方便管理员进行博客内容的管理和维护。
